# 1. 내 컴퓨터에서 앱 실행하기 (Environment Setup)

### 상황
Google AI Studio에서 만든 앱을 내 컴퓨터(로컬 환경)로 가져왔습니다.

### 배움
- **의존성 설치 (`npm install`)**: 앱이 실행되려면 여러 가지 부품(라이브러리)이 필요합니다. 이 명령어는 인터넷에서 필요한 부품들을 다 받아와서 조립할 준비를 하는 과정입니다.
- **앱 실행 (`npm run dev`)**: 개발 모드로 앱을 실행합니다. 이제 내 컴퓨터가 작은 서버가 되어 `localhost:3000` 주소로 접속할 수 있게 됩니다.
