# ✨ 인터랙티브 UX: 하이엔드 UI 애니메이션 (High-End Animations)

사이버펑크 테마의 몰입감을 극대화하기 위해 적용된 고급 애니메이션 기술들을 설명합니다.

---

### 1. 🌈 무지개 액체 테두리 (Liquid Gradient Border)

버튼 주변에 색상이 회전하며 흐르는 듯한 효과를 줍니다.
- **기술**: CSS `conic-gradient` + `framer-motion`
- **원리**: `conic-gradient`를 배경으로 가진 요소를 버튼 뒤에 배치하고, `animate` 속성을 이용해 배경의 각도를 0도에서 360도로 무한 회전시킵니다.
- **효과**: 마치 버튼 테두리에 네온 사인이 흐르는 듯한 생동감을 부여합니다.

### 2. 📡 스캐닝 라인 (Scanning Line)

시스템이 지속적으로 데이터를 읽고 있는 듯한 느낌을 줍니다.
- **기술**: `motion.div` + `absolute` positioning
- **원리**: 아주 얇은 흰색 선(또는 밝은 색)을 버튼 내부 최상단에 배치하고, `top` 좌표를 `-10%`에서 `110%`까지 반복적으로 이동시킵니다.
- **효과**: "분석 중" 또는 "작동 중"이라는 인상을 주어 유저에게 "재미"와 "신뢰"를 동시에 전달합니다.

### 3. 📐 쉐이프 동기화 (Shape Synchronization - Skew)

사이버펑크 특유의 기울어진(Skewed) 디자인을 유지하면서 애니메이션을 입히는 법입니다.
- **주의점**: 버튼 자체가 `skew-x-[-10deg]`인 경우, 그 뒤에 배치하는 글로우(Glow)나 테두리 요소도 동일한 `skew` 값을 가져야 형태가 어긋나지 않습니다.
- **디자인 팁**: 텍스트(Children)는 다시 반대로 `skew-x-[10deg]`를 주어 똑바로 보이게 하는 것이 가독성에 좋습니다. (이미 `Button` 컴포넌트에 구현됨)

### 4. 🖱️ 상호작용 피드백 (Interactive Feedback)

유저의 행동에 즉각적으로 반응하는 시각 효과들입니다.
- **Scale Up**: 마우스를 올리면 버튼이 살짝 커지며 "누를 수 있음"을 강조합니다.
- **Pulse Overlay**: 마우스를 올렸을 때 테두리가 밖으로 퍼져 나가는 애니메이션을 추가하여 클릭 욕구를 자극합니다.
- **Glow Intensity**: `blur` 값을 동적으로 조절하여 마우스를 올렸을 때 빛이 더 강하게 퍼지도록 합니다.

---

### 💡 유저 경험(UX) 포인트
단순히 예쁜 것을 넘어, **"이 버튼은 살아있고 내 조작을 기다리고 있다"**는 느낌을 주는 것이 핵심입니다. 이러한 마이크로 인터랙션은 앱의 퀄리티를 전문가 수준(High-End)으로 보이게 만듭니다.
