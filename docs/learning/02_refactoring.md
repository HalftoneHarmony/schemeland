# 2. 리팩토링: 원룸에서 아파트로 (Refactoring)

### 상황
`App.tsx` 파일 하나에 모든 코드가 다 들어있어서 파일이 너무 길고 복잡했습니다 (1200줄 이상).

### 비유: "원룸 vs 방 여러 개"
- **이전 (`App.tsx` 몰빵)**: 커다란 원룸에 침대, 부엌, 화장실, 책상이 다 섞여 있는 상태. 뭐 하나 찾으려면 온 방을 다 뒤져야 합니다.
- **이후 (컴포넌트 분리)**:
    - `App.tsx`: **거실** (전체적인 이동 통로)
    - `LandingView.tsx`: **현관** (첫인상)
    - `BrainDumpView.tsx`: **서재** (생각 정리)
    - `DashboardView.tsx`: **안방** (메인 활동 공간)
    - ...등등
    
이제 "현관을 꾸미고 싶으면" `LandingView.tsx`만 찾아가면 됩니다. 코드를 관리하기 훨씬 쉬워졌습니다!
