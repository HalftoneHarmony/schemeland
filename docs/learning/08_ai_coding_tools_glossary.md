# AI 코딩 도구 용어 정리 및 멀티 에이전트 개념 (AI Coding Tools Glossary)

비개발자 입장에서 헷갈리기 쉬운 AI 코딩 툴들의 정체와 차이점, 그리고 활용법을 정리합니다.

---

## 1. AI 코딩 도구: 누가 누구인가? (Who is Who?)

### 🤖 Antigravity (앤티그래비티)
*   **정체**: Google DeepMind가 개발한 **"Agentic AI Coding Assistant"**.
*   **포지션**: IDE(코드 편집기) 안에 상주하는 **자율 주행 개발자**.
*   **특징**: 단순히 코드만 추천하는 게 아니라, 터미널 명령어를 입력하고, 파일을 생성/수정하고, 브라우저를 열어 테스트까지 수행하는 **행동하는 AI**입니다.

### 🧠 Codex (코덱스)
*   **정체**: OpenAI가 개발한 **AI 모델의 이름**. (도구가 아니라 '엔진'입니다.)
*   **포지션**: **GitHub Copilot**의 두뇌.
*   **특징**: "자동차(Copilot)"가 아니라 자동차에 들어가는 "엔진(Codex)"입니다. 우리가 직접 Codex를 쓴다기보다, Copilot이라는 툴을 통해 사용하게 됩니다.

### 📜 Claude Code (클로드 코드)
*   **정체**: Anthropic이 개발한 **터미널 기반 AI 코딩 도구**.
*   **포지션**: 터미널(검은 화면)에 사는 경쟁사 개발자.
*   **특징**: IDE가 아니라 터미널에서 대화하며 코딩합니다. "로그인 기능 만들어줘"라고 치면 알아서 파일을 고칩니다.

### 🗺️ ChatGPT Atlas (가칭/예상)
*   **정체**: OpenAI의 Canvas 또는 차세대 코딩 인터페이스.
*   **포지션**: 채팅창과 코드 에디터가 결합된 형태의 작업 공간.
*   **특징**: 단순히 채팅만 하는 게 아니라, 오른쪽 화면에서 문서를 직접 보면서 수정하는 방식.

## 2. Antigravity와 IDE의 관계

*   **IDE (Integrated Development Environment)**: 주방 (Kitchen). 요리를 하기 위한 공간과 도구(칼, 불, 도마)가 모여 있는 곳. (예: VS Code, Cursor)
*   **Antigravity**: 요리사 (Chef). 주방에 들어와서 칼질하고 불을 쓰는 사람.
*   **결론**: Antigravity는 IDE 그 자체는 아니지만, IDE를 마음대로 조종하는 **"슈퍼 요리사 플러그인"**입니다.

---

## 3. Antigravity로 멀티 에이전트(Multi-Agent) 굴리기

**Q: 하나의 Antigravity로 여러 명의 AI를 동시에 부릴 수 있나요?**

**A: 네, "분신술"을 쓰면 됩니다.**

1.  **내장된 서브 에이전트 (Internal Sub-agents)**
    *   Antigravity는 이미 혼자가 아닙니다. 내부에 **'브라우저 담당'**, **'터미널 담당'**, **'코드 수정 담당'** 부하들을 거느리고 있습니다. 시키는 일에 따라 알아서 전문가를 파견합니다.

2.  **다중 인스턴스 (Multiple Instances) - "8인의 용병단"**
    *   사용자 입장에서 여러 명을 동시에 쓰고 싶다면, **VS Code 창을 여러 개 띄우면 됩니다.**
    *   **창 1**: UI 담당 (디자이너 AI에게 지시)
    *   **창 2**: DB 담당 (백엔드 AI에게 지시)
    *   **창 3**: QA 담당 (테스터 AI에게 지시)
    *   이렇게 하면 물리적으로 **8명의 앤티그래비티**가 동시에 내 프로젝트를 위해 일하게 됩니다. 이때 필수적인 것이 바로 앞서 배운 **`Git Branch` (평행 우주)** 전략입니다.
