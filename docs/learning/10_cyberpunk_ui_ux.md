# 09. 사이버펑크 UI/UX 디자인 시스템

비개발자도 이해할 수 있는 **SchemeLand Cyberpunk Edition**의 디자인 시스템과 구현 원리에 대한 학습 기록입니다.

## 1. 디자인 컨셉: 사이버펑크 (Cyberpunk)
사이버펑크 스타일은 고도의 기술력과 거친 도시 느낌을 결합한 미학입니다. 이를 웹 UI에 적용하기 위해 다음과 같은 핵심 요소를 사용했습니다:
- **네온 컬러 (Neon Colors)**: 어두운 배경에 대비되는 강렬한 핑크, 시안, 옐로우.
- **하이테크 타이포그래피**: 미래적인 느낌을 주는 영문 폰트.
- **디지털 노이즈 & 텍스처**: 그리드 패턴, 스캔라인(Scanlines), 글리치(Glitch) 효과.
- **기하학적 변형**: 정형화된 사각형에서 탈피한 기울어진(Skewed) 형태.

---

## 2. 디자인 시스템 구현 (Tailwind CSS)

### 핵심 컬러 팔레트
`index.html`의 테마 설정을 통해 앱 전체에서 사용할 색상을 정의했습니다.
- **Primary (Neon Pink)**: `#ff00ff` - 주요 액션 버튼 및 강조.
- **Secondary (Neon Cyan)**: `#00ffff` - 보조 정보 및 텍스트 강조.
- **Accent (Cyber Yellow)**: `#fcee0a` - 경고 또는 특별 포인트.
- **Background**: `#050505` (거의 검은색) - 네온 효과를 극대화하기 위한 심연의 배경.

### 미래지향적 폰트
- **Orbitron**: 헤드라인용. 기하학적이고 각진 형태로 "기계적인" 느낌을 줌.
- **Rajdhani**: 본문용. 가로폭이 좁고 깔끔하여 많은 정보를 세련되게 전달.
- **Pretendard**: 가독성을 위한 기본 한글 폰트.

---

## 3. 주요 UI 컴포넌트 기술

### A. 네온 글로우 버튼 (Neon Glow Buttons)
단순한 색상이 아니라, 빛이 번지는 듯한 효과를 주기 위해 `box-shadow`를 활용했습니다.
```css
/* 예시: 핑크 네온 그림자 */
box-shadow: 0 0 5px #ff00ff, 0 0 20px #ff00ff88;
```
- **효과**: 버튼이 마치 화면 위에서 빛나고 있는 듯한 입체감을 줍니다.
- **인터랙션**: 호버(마우스 올리기) 시 그림자가 더 커지거나 색상이 반전되는 애니메이션을 적용했습니다.

### B. 배경 디테일 (Background Aesthetics)
단순한 검은색 배경은 심심할 수 있습니다. 이를 보완하기 위해 두 가지 레이어를 추가했습니다.
1. **디지털 그리드 (Grid Pattern)**: `linear-gradient`를 이용해 가로세로 30px 간격의 미세한 선을 그려 모눈종이 같은 베이스를 깔았습니다.
2. **스캔라인 (Scanlines)**: 오래된 모니터에서 위아래로 움직이는 선 효과를 CSS 애니메이션(`@keyframes scanline`)으로 구현했습니다.

### C. 기하학적 변형 (Skew Effect)
버튼이나 배너에 `skew-x-[-10deg]`와 같은 속성을 주어 평행사변형 형태로 만들었습니다. 이는 정적인 디자인에 역동성과 속도감을 부여합니다.

---

## 4. 애니메이션 & 인터랙션 (Framer Motion)
고급스러운 느낌을 위해 **Framer Motion** 라이브러리를 사용했습니다.
- **패럴랙스 (Parallax)**: 스크롤 속도에 따라 배경의 빛무리(Blobs)가 서로 다른 속도로 움직여 공간감을 형성합니다.
- **블러 등장 (Blur Entrance)**: 요소가 나타날 때 단순 투명도뿐만 아니라 '흐림' 효과가 사라지며 나타나게 하여 몽환적인 느낌을 줍니다.
- **스테거 (Stagger)**: 목록 아이템들이 0.15초 간격으로 순차적으로 등장하게 하여 생동감을 불어넣었습니다.

---

## 5. 학습 포인트
- **비전의 시각화**: 어떠한 테마(사이버펑크)를 정했을 때, 단순히 색만 바꾸는 것이 아니라 폰트, 질감, 애니메이션이 모두 하나의 철학을 공유해야 완성도가 높아집니다.
- **성능 유지**: 화려한 효과(blur, animation)를 많이 쓰면 저사양 기기에서 느려질 수 있습니다. 이를 위해 `backdrop-blur`와 같은 GPU 가속을 활용하는 속성을 적절히 배치하는 것이 중요합니다.
